# 🎯 Core Product Positioning

Your biggest opportunity is:

**Simple when starting, powerful when scaling.**

The free version should help users:

* launch first course
* enroll first students
* make first sale

The paid versions should help them:

* improve outcomes
* grow revenue
* scale teams
* build marketplace businesses

# 🧲 HIGHEST CONVERTING PAYWALL MOMENTS

Use upgrade prompts at these exact product moments:

1. after first course publish → unlock drip
2. after first 10 students → unlock reports
3. when adding second instructor → unlock multi instructor
4. when enabling recurring payments → unlock subscriptions
5. when selling multiple courses → unlock bundles
6. when adding vendors → unlock marketplace
