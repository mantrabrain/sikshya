== Description ==

Sikshya is free Learning management system (LMS) for WordPress. It helps to create course, lessons, quizzes, questions and answers for your online course system. You can sell the course easily with sikshya LMS.

[Sikshya LMS](https://mantrabrain.com/downloads/sikshya-wordpress-learning-management-system/) || [Sikshya LMS Theme](https://mantrabrain.com/downloads/pragyan-wordpress-theme/)  

You can create, manage and sell online courses after building fully functional online learning website with this free learning management plugin. You can easily create courses, sections, lessons, quizzes and sell those items online without any cost.

You can Join our [Facebook Community group](https://www.facebook.com/groups/mantrabraincommunity)

= Features =

Here are some popular features of Sikshya LMS.

  * Easy course builder
  * Unlimited courses and lessons
  * Frontend student dashboard
  * Youtube video 
  * Lesson management
  * Course requirement & outcomes
  * Quizzes
  * Order Report
  * Course Widget
  * Course Duration
  * Course Difficulty Level
  * Different type of quizzes
  * Export/Import Feature

== Installation ==

1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server (in the /wp-content/plugins/ directory).
2. Activate the Mantra Audience plugin through the 'Plugins' menu in WordPress.
